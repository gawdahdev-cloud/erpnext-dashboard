
# منصة جودة التعليم - لوحة التحكم

هذا المشروع هو لوحة تحكم (Dashboard) مبنية باستخدام React + Vite، مصممة لعرض إحصائيات من مؤسسات تعليمية متعددة تعمل بنظام Frappe ERP.

## الميزات الرئيسية

- **مصادقة آمنة:** يستخدم بروتوكول OAuth 2.0 عبر Frappe "Connected App" لضمان تسجيل دخول آمن دون كشف أي بيانات سرية في الواجهة الأمامية.
- **عرض موحد:** يقدم نظرة شاملة على مؤشرات الأداء الرئيسية (KPIs) لجميع المؤسسات من مكان واحد.
- **أداء محسن:** تم تطبيق تحسينات متقدمة لضمان سرعة التحميل والأداء:
  - **التحميل المتأخر (Lazy Loading):** لا يتم تحميل المكونات إلا عند الحاجة إليها.
  - **تقسيم الكود (Code Splitting):** يتم فصل المكتبات الكبيرة (vendor chunk) لتقليل حجم الملف الأولي.
  - **تحليل الحزمة (Bundle Analysis):** استخدام `rollup-plugin-visualizer` لتحليل وفهم حجم الملفات النهائية.

## 🚀 طريقة الإعداد والتشغيل

### 1. إعداد Frappe Connected App

قبل تشغيل المشروع، يجب عليك إعداد "Connected App" في نظام Frappe ERP الخاص بـ **المركز**.

1.  اذهب إلى واجهة Frappe ERP.
2.  ابحث في الشريط عن **"Connected App"** وانتقل إلى القائمة.
3.  أنشئ سجلاً جديداً (`New`).
4.  املأ البيانات التالية:
    - **App Name**: `dashboard` (أو أي اسم تفضله).
    - **Scopes**: حدد `all` للوصول الكامل.
    - **Redirect URIs**: أضف الرابط التالي:
      ```
      http://localhost:3000
      ```
      *(إذا كنت تخطط لنشر التطبيق، أضف رابط الموقع النهائي هنا أيضاً)*.
5.  بعد الحفظ، سيقوم Frappe بإنشاء **Client ID**. انسخ هذا الـ ID.

### 2. إعداد المشروع المحلي

1.  **نسخ المشروع:**
    ```bash
    git clone <رابط-المستودع>
    cd <اسم-المجلد>
    ```

2.  **تثبيت الاعتماديات:**
    ```bash
    npm install
    ```

3.  **إضافة Client ID:**
    - افتح ملف `vite.config.js`.
    - ابحث عن المتغير `OAUTH_CLIENT_ID` في أعلى الملف.
    - الصق الـ **Client ID** الذي نسخته من Frappe.

    ```javascript
    // vite.config.js
    export const OAUTH_CLIENT_ID = 'هنا_الصق_الـ_CLIENT_ID';
    ```

### 3. تشغيل المشروع

- لبدء خادم التطوير:
  ```bash
  npm run dev
  ```
- لعمل build للنسخة النهائية:
  ```bash
  npm run build
  ```
  بعد عملية الـ build، سيتم إنشاء ملف `stats.html` يوضح خريطة حجم الحزمة النهائية.
